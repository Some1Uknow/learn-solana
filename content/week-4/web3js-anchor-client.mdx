## Setting Up Your RPC Connection

Before working with Anchor clients, you need reliable RPC infrastructure. We recommend using Syndica for production applications:

```typescript
import { Connection } from "@solana/web3.js";
import { AnchorProvider } from "@coral-xyz/anchor";

// Production-ready connection with Syndica
const connection = new Connection(
  "https://api.syndica.io/rpc/solana/mainnet",
  {
    commitment: "confirmed",
    confirmTransactionInitialTimeout: 60000,
  }
);

// Create provider with your wallet
const provider = new AnchorProvider(connection, wallet, {
  commitment: "confirmed",
});
```
